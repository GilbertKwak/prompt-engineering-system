## 🎯 라이브러리 목적

모든 **신사업·제품 기획 논의**를 단일 MECE 프레임워크로 통일하여,
전략·재무·기술 분석의 출력 포맷을 자동 표준화합니다.

> **핵심 원칙**: 어떤 신사업 아이디어, 제품 로드맵, 시장 진입 전략이든
> PE-PROD 템플릿을 통과하면 동일한 구조·포맷·품질 기준으로 산출됩니다.
