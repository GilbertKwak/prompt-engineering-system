## 3. v8.0 신규 Job 스켈레톤

### 3-1. Job 2.5: EW Enrichment (신규)

```yaml
# ─── Job 2.5: EW 신호 상세 분석 ────────────────────────────────────────────
ew-enrichment:
  name: "2.5. EW 신호 상세 분석 (v8.0 NEW)"
  needs: [protocol-check, ew-detection]
  runs-on: ubuntu-latest
  if: needs.ew-detection.outputs.ew_triggered == 'true'

  outputs:
    enrichment_summary: ${{ steps.enrich.outputs.summary }}
    enrichment_json: ${{ steps.enrich.outputs.json_path }}

  steps:
    - uses: actions/checkout@v4
    - uses: actions/setup-python@v5
      with:
        python-version: ${{ env.PYTHON_VERSION }}
        cache: "pip"
    - run: pip install -r automation/requirements_ai_intel.txt

    - name: EW 신호 상세 분석
      id: enrich
      env:
        PERPLEXITY_API_KEY: ${{ secrets.PERPLEXITY_API_KEY }}
        EW_SIGNALS: ${{ needs.ew-detection.outputs.ew_signals }}
        EW_SEVERITY: ${{ needs.ew-detection.outputs.ew_severity }}
      run: |
        python automation/ai_ew_enricher.py \
          --signals "$EW_SIGNALS" \
          --severity "$EW_SEVERITY" \
          --week "${{ needs.ew-detection.outputs.week }}" \
          --output "output/ai_intel/ew_enriched.json"
        # TODO: outputs 파싱 추가
```

### 3-2. Job 3.5: KG Commit (신규)

```yaml
# ─── Job 3.5: KG 스냅샷 자동 커밋 ──────────────────────────────────────────
kg-commit:
  name: "3.5. KG 스냅샷 자동 커밋 + PR (v8.0 NEW)"
  needs: [kg-delta]
  runs-on: ubuntu-latest

  steps:
    - uses: actions/checkout@v4
      with:
        token: ${{ secrets.GITHUB_TOKEN }}

    - name: KG 아티팩트 다운로드
      uses: actions/download-artifact@v4
      with:
        name: kg-delta
        path: knowledge_graphs/

    - name: KG 스냅샷 커밋
      run: |
        git config user.name  "github-actions[bot]"
        git config user.email "github-actions[bot]@users.noreply.github.com"
        git add knowledge_graphs/
        git diff --cached --quiet || git commit -m "chore: KG 스냅샷 v${{ needs.kg-delta.outputs.kg_next_version }} (${{ needs.ew-detection.outputs.week }})"
        git push
```

### 3-3. Job 6.5: Slack Notify (신규)

```yaml
# ─── Job 6.5: Slack 알림 ────────────────────────────────────────────────────
slack-notify:
  name: "6.5. Slack 파이프라인 알림 (v8.0 NEW)"
  needs: [protocol-check, ew-detection, kg-delta, pipeline-summary]
  runs-on: ubuntu-latest
  if: always() && env.SLACK_NOTIFY_ON_EW == 'true'

  steps:
    - name: Slack 웹훅 전송
      env:
        SLACK_WEBHOOK_URL: ${{ secrets.SLACK_WEBHOOK_URL }}
      run: |
        EW="${{ needs.ew-detection.outputs.ew_triggered }}"
        SEV="${{ needs.ew-detection.outputs.ew_severity }}"
        WEEK="${{ needs.ew-detection.outputs.week }}"
        KG_VER="${{ needs.kg-delta.outputs.kg_next_version }}"

        EMOJI="✅"
        if [ "$EW" = "true" ] && [ "$SEV" = "HIGH" ]; then EMOJI="🚨"; fi
        if [ "$EW" = "true" ] && [ "$SEV" = "MEDIUM" ]; then EMOJI="⚠️"; fi

        curl -s -X POST "$SLACK_WEBHOOK_URL" \
          -H "Content-Type: application/json" \
          -d "{
            \"text\": \"${EMOJI} PE AI Intel Weekly — ${WEEK}\",
            \"attachments\": [{
              \"color\": \"good\",
              \"fields\": [
                {\"title\": \"EW 상태\", \"value\": \"${EW} (${SEV})\", \"short\": true},
                {\"title\": \"KG 버전\", \"value\": \"v${KG_VER}\", \"short\": true}
              ]
            }]
          }"
```
