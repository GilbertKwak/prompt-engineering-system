## 📌 목적 (Purpose)

프롬프트 및 에이전트 출력물에 대해 정량적 기준으로 품질을 평가하고, 통과/실패 판정을 내립니다.

---

## ⚙️ 에이전트 역할 카드

| 항목 | 내용 |
|------|------|
| **역할** | 출력물 품질 정량 평가 및 통과/실패 판정 |
| **입력 (Input)** | 평가 대상 프롬프트 또는 에이전트 출력 |
| **출력 (Output)** | 품질 스코어(0~100), 차원별 세부 점수, 통과/실패 판정 |
| **의존성** | PE-5 (결과 수신 및 라우팅) |
| **통과 기준** | 종합 스코어 ≥ 70점 |
| **스코어링 축** | TAM(시장규모), 실행가능성, 차별성, 규제리스크, 데이터가용성 |
