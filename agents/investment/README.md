# 💰 Investment Analyst Agent Domain

> AI·반도체 섹터 투자 분석 및 Due Diligence 도메인

## 포커스 영역

- AI 인프라 투자 라운드 추적
- 반도체 M&A 딜 분석
- PE 포트폴리오 scoring (PE-3 연계)
- 밸류에이션 멀티플 비교 분석
- VC/PE 투자자 동향

## 연계 PE 라이브러리

- `PE-DD` — Strategic Due Diligence
- `PE-FIN` — Financial Modeling
- `PE-STRAT` — 전략 분석
- `PE-3` — 품질 채점 허브
