# v4.0 — Hyperscaler / Corporate VC 전략 커스텀 프롬프트

> **작성일**: 2026-03-01  
> **프롬프트 유형**: 전략적 LP 타겟 커스텀  
> **적용 프로젝트**: AstraChips AI Infrastructure Fund I

## 📥 프롬프트 원문

```
특정 LP(예: hyperscaler CVC, corporate VC 등)를 겨냥한
커스텀 버전으로도 재구성
```

## 🎯 핵심 지시사항
1. 순수 재무 LP가 아닌 **전략적 LP**(하이퍼스케일러, 반도체·클라우드 기업) 대상
2. 내부 CAPEX를 보완하는 **"외부 확장 팔(Extended arm)"** 포지셔닝
3. Co-investment + Capacity Reservation + Control Option 3가지 협업 모드 제시
4. **한국어·영어 각각 별도 완전 문서로 작성**

## 🔧 전략 파라미터
```yaml
collaboration_modes:
  1_forward_capacity:
    - ROFO (Right of First Offer)
    - Volume Commitment 구조
    - Priority Allocation (전용 라인 옵션)
  2_strategic_co_invest:
    - 개선된 fee/carry 조건 제공
    - 직접 공동 투자 (Korea Glass+HBM JV, Taiwan CoWoS, UAE SMR)
  3_control_options:
    - Call option / Step-in right
    - Buy-up mechanism (내재화 경로)

governance_for_strategic_lp:
  - Technical Steering Committee 참여
  - Capacity ramp + Yield trend 리포팅
  - Confidentiality & Conflict management framework

off_balance_sheet_benefit: true
capacity_access: true
contact: partners@astrachips-fund.com
```

## 📐 섹션 구조
```
1. Strategic Fit → 외부 확장 팔 포지셔닝
2. Why This Matters to You → Capacity bottleneck Pain Point
3. Co-Design / Co-Investment Model → 3개 협업 모드 상세
4. Technical Focus Areas → HBM·Packaging + Power·Thermal
5. Governance & Information Rights → Steering Committee, 맞춤 리포팅
6. Financial Profile + Strategic Optionality → Off-BS + 업사이드
```
