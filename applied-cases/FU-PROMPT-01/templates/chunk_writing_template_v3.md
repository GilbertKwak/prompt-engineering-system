## TIER별 프롬프트 호출 가이드

### TIER-1 호출 (1~4 섹션)
```
FU-PROMPT-01-T1 호출
대상 섹션: [S01, S02, S03]
파라미터북: [경로]
```

### TIER-2 호출 (5~9 섹션 | Chunk 분할 필수)
```
FU-PROMPT-01-T2 호출
Chunk: A
대상 섹션: [S01, S02, S03]
파라미터북: [경로]
이전 Chunk 핵심 수치: [없음 / {prev_summary}]
```

### TIER-3 호출 (10+ 섹션 | 레지스트리 필수)
```
FU-PROMPT-01-T3 호출
Chunk: A (1/{total_chunks})
대상 섹션: [S01, S02]
전체 섹션 맵: [S01~S13]
파라미터북: [경로]
레지스트리: [없음 / {prev_registry}]
QA: STRICT
```
