# 🤖 PE-5 — 마스터 오케스트레이터

> **ID**: PE-5 | **상태**: 🟡 개발 중

---

## 역할

PE-1~4를 통합 조율하는 상위 엔진입니다. 사용자 의도 파악, 에이전트 라우팅, 결과 통합을 담당합니다.

## 설계 원칙

- **Scope 명확화**: 각 엔진의 역할 경계 준수
- **중복 방지**: Orchestrator만 요약을 생성, 하위 엔진은 세부 결과만 출력
- **실패 처리**: 엔진 실패 시 자동 재시도 (max 2회)
