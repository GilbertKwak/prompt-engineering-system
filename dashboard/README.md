# 📊 Dashboard — 시스템 메트릭스 대시보드

> **최종 업데이트**: 2026-04-09 | **버전**: v1.0

---

## 개요

프롬프트 엔진 시스템의 핵심 KPI와 RCA 구현 효과를 한눈에 볼 수 있는 수치 대시보드입니다.

## 구성 파일

| 파일 | 내용 |
|------|------|
| [metrics.md](./metrics.md) | KPI 추적 시트 — 전체 지표 현황 |

## 활용 가이드

- 매주 금요일: `metrics.md` 확인 및 신규 실행 입력
- 월말: 지났 달 성과 평가 및 다음 달 목표 설정
- 지표가 법위를 벗어나면: 신규 RCA 선로에 등록 (`docs/rca-capa/`)
