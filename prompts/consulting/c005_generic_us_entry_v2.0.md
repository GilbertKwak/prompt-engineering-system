GTM 예산 (연): {{gtm_budget}}
핵심 경쟁 우위: {{competitive_advantage}} (1~3가지)

[TASK]
다음 4개 영역으로 미국 시장 진출 전략을 수립하라:

1. 시장 진입 준비도 진단 (Market Readiness Assessment)
   - 제품·서비스의 미국 시장 적합성: PMF(Product-Market Fit) 가설 3개
   - 현지화 필요 요소: 언어 / 규제 / 문화 / 영업 방식
   - 진입 준비도 점수 (0~100) + 보완 필요 영역 TOP 3
   - 적정 진입 타이밍 권고 (즉시 / 3개월 후 / 6개월 후) + 근거

2. 진출 모델 선택 (Market Entry Model)
   - 옵션 A: 직접 진출 (법인 설립 + 현지 영업팀)
   - 옵션 B: 파트너십 (OEM / VAR / 에이전트)
   - 옵션 C: 디지털 퍼스트 (온라인 셀프서브 + 인바운드)
   - 각 옵션: 초기 투자 / 손익분기 시점 / 적합한 기업 유형
   - {{company_name}}에 최적 옵션 1개 선정 + 선정 근거

3. 첫 12개월 실행 계획
   - Q1 (Month 1~3): 기반 구축 — 핵심 액션 5개 + 담당자 역할
   - Q2 (Month 4~6): 첫 고객 확보 — 목표 파이프라인 규모 + KPI
   - Q3 (Month 7~9): 스케일업 기반 — 레퍼런스 확보 + 채용 계획
   - Q4 (Month 10~12): 성과 검증 — ARR 목표 대비 달성률 기준

4. 성공 지표 & 실패 조기 경보
   - 성공 지표 (North Star Metric 1개 + 보조 지표 3개)
   - 실패 조기 경보 신호 3개 + 각 신호별 대응 트리거
   - 6개월 시점 Go/Pivot/Stop 판단 기준

[OUTPUT FORMAT]
- 진입 준비도는 점수판 + 레이더 차트 텍스트 버전
- 진출 모델 비교는 표(Table) 형식
- 실행 계획은 분기별 카드 형식
- 총 분량: 1,500~2,500자

□ 진입 준비도 점수에 근거(강점/약점)가 명시되었는가?
□ 진출 모델 비교표가 3개 옵션을 모두 포함하는가?
□ Q1~Q4 실행 계획이 각각 구체적 액션과 KPI를 포함하는가?
□ 위 4개 중 하나라도 NO이면 해당 섹션 재작성
