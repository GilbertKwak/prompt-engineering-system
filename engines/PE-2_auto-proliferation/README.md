# 🌱 PE-2: 자동증식 엔진 (Auto-Proliferation Engine)

> **Parent**: [prompt-engineering-system](https://github.com/GilbertKwak/prompt-engineering-system)  
> **버전**: v1.1 | **최종 수정**: 2026-04-05

---

## 개요

검증된 단일 프롬프트로부터 **목적·도메인·난이도·포맷**을 달리한  
다수의 변형 버전을 자동으로 생성하는 엔진입니다.

---

## 증식 매트릭스

| 변형 축 | 변형 옵션 |
|---------|-----------|
| **목적** | 분석 / 요약 / 보고서 작성 / 의사결정 지원 / 교육 |
| **도메인** | 반도체 / 에너지 / 금융 / 의료 / 일반 |
| **난이도** | 초급 / 중급 / 전문가 |
| **포맷** | 서술형 / 표 / 불릿 / JSON / 슬라이드 |
| **언어** | 한국어 / 영어 / 이중언어 |

**최대 생성 수**: 5×5×3×5×3 = 1,125 변형 (실용 목적: 3~10개 선택)

---

## 작동 원리

```
Step 1: 시드 프롬프트 (PE-1 통과 버전) 입력
Step 2: 증식 매트릭스에서 목표 조합 선택
Step 3: 각 조합에 맞춰 자동 변형 생성
Step 4: 변형 버전에 PE-3 자동검증 적용
Step 5: 합격 버전만 라이브러리 등록
```
