## ⚙️ PE-3 자동검증 프로세스 v2.0

```
[INPUT] 프롬프트 또는 결과물

검증 레이어 1: Pearl SCM
  - 인과 그래프 추출: X → Y (직접 인과?)
  - do-calculus: P(Y|do(X)) ≠ P(Y|X)인 경우 오류 플래그
  - 반사실 추론: "X=0이었다면 Y는?"
  - 교란변수 탐지

검증 레이어 2: Feynman 제1원리
  - 핵심 주장 추출
  - "이 주장의 물리적/논리적 근거는?"
  - 순환 논리 탐지
  - 추상화 레이어 과다 여부 확인

검증 레이어 3: Ng 데이터 검증
  - 측정 가능성: 지표 존재 여부
  - 재현성: 동일 입력 → 동일 출력 가능?
  - 샘플 크기 / 데이터 품질 검토

검증 레이어 4: Porter 전략 검증
  - 산업 맥락 적합성
  - 경쟁 구조 반영 여부
  - 가치사슬 위치 확인

검증 레이어 5: Hinton 종합 판정
  - 4개 레이어 결과 가중 통합
  - w = [Pearl:0.35, Feynman:0.30, Ng:0.20, Porter:0.15]
  - 최종 PE-3 점수 산출

Bayesian 신뢰도 출력:
  Prior: Beta(α=2, β=9)
  P(검증 통과) = [계산값]
  임계값: P > 0.85 → PASS / P < 0.85 → REVISE

PE-3 출력 형식:
  검증 결과: PASS / REVISE / FAIL
  오류 목록: [있다면 열거]
  PE-3 Score: [0-100]
  Bayesian P: [0-1]
  권고 조치: [있다면 명시]
```
