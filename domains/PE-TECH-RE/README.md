## Domain Description

PE-TECH-RE는 어떤 아티팩트든 그것을 생성한 **인지 아키텍처**를 역설계(Reverse Engineering)하는 데 특화된 프롬프트 도메인입니다.
표면적 콘텐츠가 아닌 **인지 구조, 계획 깊이, 메타추론, 에이전트 생태계**를 복원하는 것이 핵심 목적입니다.
