## 📌 목적 (Purpose)

## ⚙️ 에이전트 역할 카드

| 항목 | 내용 |
|------|------|
| **역할** | 실적용 케이스 실행 및 결과 아카이빙 |
| **입력 (Input)** | 검증된 프롬프트(PE-3 통과), 적용 도메인 컨텍스트 |
| **출력 (Output)** | 케이스 실행 리포트, `applied-cases/` 파일 생성 |
| **의존성** | PE-3 (검증 통과), PE-5 (오케스트레이션) |
| **저장 경로** | `applied-cases/YYYY-MM-DD_케이스명.md` |
| **개발 상태** | 기본 구조 설계 완료, 자동화 파이프라인 미완 |
