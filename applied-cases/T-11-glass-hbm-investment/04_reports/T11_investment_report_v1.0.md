**4대 정량 지표 × 가중치 체계**

```python
# VS_portfolio 산출 공식
VS_portfolio = (
    W1 * supply_concentration_score   # 공급 집중도
  + W2 * replaceability_score         # 대체 가능성 역수
  + W3 * geopolitical_exposure_score  # 지정학 노출도
  + W4 * policy_uncertainty_score     # 정책 불확실성
)
# W1=0.30, W2=0.25, W3=0.30, W4=0.15
# 정상 범위: 0.5 ~ 2.5 | 경보 임계: > 3.0 | 위험 임계: > 4.0
```
