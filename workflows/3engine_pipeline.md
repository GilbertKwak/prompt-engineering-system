## 파이프라인 흐름도

```
[INPUT] 사용자 초안 프롬프트
        │
        ▼
┌───────────────────────┐
│   PE-1: 자동개선       │◄──── 재처리 루프 (max 3회)
│   - 7항목 진단         │
│   - 타겟 재작성        │
│   - 품질 비교표 출력   │
└──────────┬────────────┘
           │ 6/7 이상 합격
           ▼
┌───────────────────────┐
│   PE-2: 자동증식       │
│   - 변형 매트릭스 선택 │
│   - N개 버전 생성      │
│   - 각 버전 메타데이터 │
└──────────┬────────────┘
           │ N개 변형 버전
           ▼
┌───────────────────────┐
│   PE-3: 자동검증       │
│   - 5차원 채점         │
│   - 합격/재처리 판정   │
└──────────┬────────────┘
           │
    ┌──────┴──────┐
    │             │
  Pass          Fail
    │             │
    ▼             ▼
[라이브러리 등록] [PE-1 재처리]
[OUTPUT]
```

## 실행 시간 기준

| 단계 | 단순 프롬프트 | 복잡 프롬프트 |
|------|-------------|-------------|
| PE-1 (1회) | ~2분 | ~5분 |
| PE-2 (5변형) | ~3분 | ~8분 |
| PE-3 (5건) | ~5분 | ~10분 |
| **전체** | **~10분** | **~23분** |

---

## 품질 게이트 (Quality Gates)

| 게이트 | 위치 | 합격 기준 | 실패 시 |
|--------|------|----------|--------|
| QG-1 | PE-1 출력 후 | 7항목 중 6개 이상 ✅ | PE-1 재실행 |
| QG-2 | PE-2 출력 후 | 각 변형 버전 명확히 구분 | 증식 재실행 |
| QG-3 | PE-3 출력 후 | 총점 85점 이상 | PE-1부터 재시작 |

---

## 적용 도메인 우선순위

1. **반도체 공급망 분석** (기존 적용 완료)
2. **HBM 열관리 전략** (PE-4 완전판 참조)
3. **sCO₂ 에너지 시스템** (예정)
4. **AI 데이터센터 인프라** (예정)
