# PE-3 5차원 채점 기준표 (Scoring Rubric)

## 차원 1: 정확성 (Accuracy) — 25점

| 점수 | 기준 |
|------|------|
| 23-25 | 모든 수치·사실·인용이 검증 가능하며 오류 없음 |
| 18-22 | 경미한 오류 1~2개, 핵심 내용은 정확 |
| 12-17 | 오류 3개 이상 또는 핵심 수치 오류 |
| 0-11 | 심각한 사실 오류, 즉시 재처리 필요 |

## 차원 2: 완결성 (Completeness) — 20점

| 점수 | 기준 |
|------|------|
| 18-20 | 요청된 모든 항목 완전 포함 |
| 14-17 | 선택적 항목 1~2개 누락, 필수 항목 완전 |
| 8-13 | 필수 항목 일부 누락 |
| 0-7 | 핵심 항목 다수 누락 |

## 차원 3: 구조성 (Structure) — 20점

| 점수 | 기준 |
|------|------|
| 18-20 | 명확한 계층 구조, 일관된 포맷, 스캔 용이 |
| 14-17 | 구조 대체로 명확, 소규모 포맷 불일치 |
| 8-13 | 구조 혼재, 가독성 저하 |
| 0-7 | 구조 부재, 무질서한 텍스트 |

## 차원 4: 실행가능성 (Actionability) — 20점

| 점수 | 기준 |
|------|------|
| 18-20 | 즉시 실행 가능한 구체적 다음 단계 포함 |
| 14-17 | 일반적 방향성 제시, 구체성 부분 부족 |
| 8-13 | 방향성만 있고 구체적 행동 방안 없음 |
| 0-7 | 추상적 서술만 있어 실행 불가 |

## 차원 5: 일관성 (Consistency) — 15점

| 점수 | 기준 |
|------|------|
| 13-15 | 내부 모순 없음, 용어·수치 일관성 완전 |
| 10-12 | 경미한 표현 불일치 |
| 5-9 | 내부 모순 2개 이상 |
| 0-4 | 심각한 내부 모순, 신뢰성 훼손 |

---

## 종합 판정표

| 총점 | 판정 | 후속 조치 |
|------|------|----------|
| 85-100 | ✅ Pass | 라이브러리 등록 |
| 70-84 | ⚠️ Conditional Pass | 약점 명시 후 조건부 사용 |
| 0-69 | ❌ Fail | PE-1 재처리 후 전체 파이프라인 재실행 |
